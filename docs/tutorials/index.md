## Before You Begin

Make sure you have:

- [ ] RMT Compose running (local or [live demo](https://rmt.world))
- [ ] Read [Core Concepts](/getting-started/concepts)
- [ ] Familiarized yourself with the [Workspace](/user-guide/interface/workspace)

## Tutorial Format

Each tutorial includes:

- **Objective**: What you'll build
- **Prerequisites**: What you need to know
- **Steps**: Detailed instructions
- **Verification**: How to check your work
- **Exercises**: Practice challenges
- **Next Steps**: Where to go from here

## Tips for Learning

1. **Follow along actively** - Don't just read; do each step
2. **Experiment** - Try variations on the examples
3. **Listen** - Use playback to hear your work
4. **Save frequently** - Keep snapshots of your progress
5. **Ask questions** - Use the Variable Widget to inspect values
